```python
def inclusive_sum(a, b):
    """
    Returns the inclusive sum of integers from a to b.
    
    Args:
        a (int): The start of the range (inclusive).
        b (int): The end of the range (inclusive).
    
    Returns:
        int: The sum of integers from a to b.
    
    Raises:
        ValueError: If a is greater than b.
    """
    if a > b:
        raise ValueError("a must be less than or equal to b")
    
    # Using the formula for the sum of an arithmetic series
    n = b - a + 1
    return n * (a + b) // 2
```
