```python
def parse_user(data: dict) -> dict:
    """
    Parse user data from a dictionary.

    Args:
        data (dict): Dictionary containing 'name' and 'age' keys.

    Returns:
        dict: Parsed user data with 'name' and 'age' keys.

    Raises:
        ValueError: If the input data is invalid.
    """
    if not isinstance(data, dict):
        raise ValueError("Input data must be a dictionary")

    required_keys = ['name', 'age']
    if not all(key in data for key in required_keys):
        raise ValueError("Input data must contain 'name' and 'age' keys")

    try:
        name = data['name'].strip()
        age = int(data['age'])
    except ValueError as e:
        raise ValueError("Invalid age value") from e

    return {'name': name, 'age': age}
```
Note that this refactored code includes input validation, try-except blocks, and type hints. It also raises a `ValueError` exception with a meaningful error message if the input data is invalid.
